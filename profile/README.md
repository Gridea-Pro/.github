---

Gridea Pro 是 [Gridea](https://github.com/getgridea/gridea)（10k+ ⭐）的完全重构版本，基于 **Go + Wails v2 + Vue 3** 全新打造，带来更好的性能、现代化架构和精致的写作体验。

### ✨ 亮点

- **跨平台** — 原生支持 macOS、Windows 和 Linux
- **多模板引擎** — Jinja2、EJS、Go Templates
- **一键发布** — 部署到 GitHub Pages、Vercel、Netlify 等
- **精美主题** — 持续增长的社区主题库
- **Markdown 优先** — 全功能编辑器，实时预览
- **开源免费** — GPL v3.0 协议，永久免费
