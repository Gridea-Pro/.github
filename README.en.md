Gridea Pro is a complete rewrite of [Gridea](https://github.com/getgridea/gridea) (10k+ ⭐), rebuilt from the ground up with **Go + Wails v2 + Vue 3** for better performance, modern architecture, and a polished writing experience.

### ✨ Highlights

- **Cross-platform** — Native apps for macOS, Windows, and Linux
- **Multiple template engines** — Jinja2, EJS, and Go Templates
- **One-click publishing** — Deploy to GitHub Pages, Vercel, Netlify, and more
- **Beautiful themes** — Growing collection of community themes
- **Markdown-first** — Full-featured editor with live preview
- **Open source** — GPL v3.0, free forever
